## Simple Media Mixer

Скрипт для автоматического создания итогового видео из набора фотографий и видеороликов с добавлением фоновой музыки.  
Работает на основе [MoviePy](https://github.com/Zulko/moviepy).

---

### Возможности

- Объединяет фото и видео из указанной папки в одно итоговое видео
- Поддерживает форматы: `.mp4`, `.mov`, `.avi`, `.mkv`, `.jpg`, `.jpeg`, `.png`
- Файл `000.jpg` (если есть) всегда идёт первым
- Фото автоматически масштабируются с сохранением пропорций
- Добавляется фоновая музыка (`music.mp3` по умолчанию)
- Музыка автоматически приглушается, если в видео есть собственный звук
- Все фото помещаются на фон нужного размера (например, 1920×1080)
- Финальный результат сохраняется в `final_mix.mp4`

---

###  Настройки

Все параметры задаются в начале скрипта:

```python
MEDIA_DIR = "."                  # Папка с исходными файлами
OUTPUT_FILE = "final_mix.mp4"    # Имя итогового видео
AUDIO_FILENAME = "music.mp3"     # Имя файла с фоном
IMAGE_DURATION = 3               # Длительность показа фото (сек)
MUSIC_VOLUME = 0.25              # Громкость фоновой музыки
DUCK_VOLUME = 0.5                # Громкость при наличии звука в клипе
TARGET_SIZE = (1920, 1080)       # Размер итогового кадра
BG_COLOR = (0, 0, 0)             # Цвет фона (RGB)
```

### Установка
Установите зависимости:
```
pip install moviepy numpy
```

Поместите скрипт и все медиафайлы в одну папку
```
python3 mixer.py
```  

### Принцип работы
Скрипт ищет в папке все медиафайлы (видео и изображения).
Если есть 000.jpg(ну я такой выбрала =) пишите под себя, по коду понятно, он ставится первым, остальные перемешиваются случайно.

Фото масштабируются под указанный размер кадра и размещаются по центру.

Все клипы объединяются в один ролик.

Если найден файл музыки, он добавляется и синхронизируется по длине.

При наличии аудио в видеоклипах фоновая музыка приглушается.

Итог сохраняется в final_mix.mp4.

### Совместимость
Python: 3.8+

MoviePy: 1.0+ (совместим с MoviePy 2.x)

Работает на Windows, macOS и Linux

### Возможные проблемы
Если MoviePy выдаёт предупреждение про ImageMagick, убедитесь, что он установлен, или используйте только видео и JPEG/PNG.

Для корректной работы с аудио убедитесь, что установлен ffmpeg:
```
sudo apt install ffmpeg
```
Если видео не создаётся, проверьте кодеки (libx264, aac) в вашей сборке FFmpeg.
